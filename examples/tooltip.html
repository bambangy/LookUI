Tooltip

lkTooltip(el, opts) — positioned tooltips with show/hide control

Live Demo

HTML

<span id="info-icon">
  <button class="lk-btn">Hover me</button>
</span>

JavaScript

const tip = Look.lkTooltip('#info-icon', {
  content: 'More information',
  position: 'top',  // top | bottom | left | right
});

tip.content = 'Updated text';
tip.show();
tip.hide();
tip.destroy();
