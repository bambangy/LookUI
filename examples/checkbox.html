Checkbox

lkCheckbox(el, opts) — wraps <input type="checkbox">

Live Demo

HTML

<input type="checkbox" id="agree" />

JavaScript

const agree = Look.lkCheckbox('#agree', {
  label: 'I agree to terms',
  name: 'agree',
  checked: false,
});

agree.checked;         // false
agree.toggle();        // flip state
agree.value;           // same as checked (true)
agree.setError('You must agree');
agree.clearError();
agree.destroy();
