Runtime & build-time token overrides

Scoped override — Teal brand

This block overrides --lk-primary to teal via inline CSS. No rebuild needed.

Scoped override — Pink brand

Default Badge Pink Chip

Method 1 — Runtime override (no rebuild)

<!-- Global -->
<style>:root { --lk-primary: #00897b; }</style>

<!-- Scoped -->
<div style="--lk-primary: #e91e63;">
  <button class="lk-btn lk-btn--primary">Pink</button>
</div>

Method 2 — Build-time override

/* src/scss/tokens/custom/_color.scss */
:root {
  --lk-primary:          #00897b;
  --lk-primary-dark:     #00695c;
  --lk-primary-contrast: #ffffff;
}

/* Then: npm run build:css */
