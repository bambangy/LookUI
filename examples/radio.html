Radio

lkRadio(el, opts) — wraps <input type="radio">

Live Demo

HTML

<input type="radio" id="opt-a" name="choice" />
<input type="radio" id="opt-b" name="choice" />

JavaScript

const optA = Look.lkRadio('#opt-a', { label: 'Option A', name: 'choice' });
const optB = Look.lkRadio('#opt-b', { label: 'Option B', name: 'choice' });

optA.checked;          // boolean
optA.value;            // same as checked
optA.destroy();
