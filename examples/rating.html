Rating

lkRating(el, opts) — click-to-rate star component

Live Demo

Interactive (click stars)

Read-only

Large size

HTML

<div id="stars"></div>
<!-- Stars auto-created if container is empty -->

JavaScript

const rating = Look.lkRating('#stars', {
  max: 5,
  value: 3,
  symbol: '★',
  onChange(v) { console.log('Rated:', v); },
});

rating.value;           // 3
rating.value = 4;
rating.readonly = true;
rating.destroy();
