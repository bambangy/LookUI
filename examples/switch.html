Switch

lkSwitch(el, opts) — toggle switch from <input type="checkbox">

Live Demo

HTML

<input type="checkbox" id="dark-mode" />

JavaScript

const darkMode = Look.lkSwitch('#dark-mode', {
  label: 'Dark mode',
  name: 'darkMode',
  checked: false,
});

darkMode.checked;       // false
darkMode.toggle();      // flip
darkMode.enabled = false;
darkMode.destroy();
